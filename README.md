# Panel de Administración

## Descripción
Panel de administración desarrollado como prueba técnica utilizando PHP, JavaScript, HTML y CSS.

## Tecnologías utilizadas
- PHP
- HTML5
- CSS3
- JavaScript

## Funcionalidades
- Administración de información mediante interfaz web.
- Organización modular del código.
- Interacción dinámica con JavaScript.

## Instalación
1. Clonar el repositorio.
2. Copiar la carpeta al servidor local (XAMPP, Laragon, etc.).
3. Iniciar Apache.
4. Acceder desde el navegador.
